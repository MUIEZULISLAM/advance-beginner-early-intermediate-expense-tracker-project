 Expense Category Analyzer

This is a beginner-to-intermediate Python project that analyzes expenses
based on categories using file handling and dictionaries.

## Features
- Add expenses with categories
- View all expenses
- Calculate total expenses
- Category-wise expense calculation
- Identify highest spending category

## Concepts Used
- File handling
- Dictionaries
- Loops
- Functions
- Conditional statements

## File Used
- expenses.txt

This project helped me improve my logical thinking and data aggregation skills in Python.
